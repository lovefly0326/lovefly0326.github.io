odoo12 视图挂件

tree视图

export="1"

导出详情，默认隐藏

direct_export="0"

直接导出，默认显示

editable

可编辑的tree

向导

  • 菜单动作：context
    • hide_all_button 隐藏所有向导按钮
    • hideTree2 隐藏向导按钮
      向导动作：context
    • change_color 着重按钮
    • on_tree='2' 隐藏向导按钮

odtree

no_edit_no_open='1'

用于x2many，禁止行点

line_number=’1’

行号

自定义筛选field_filter


fixed_col

固定列

form视图

标记挂件 web_ribbon


not_editable、not_deletable、no_copiable

根据domain判定是否可编辑、删除、复制

form_field_click

字段点击事件

range_field、range_related_field

字段范围挂件

form_view_initial_mode

控制form界面编辑/只读状态
可选值 edit/readonly

check_before_save、check_before_action

按钮执行前的校验逻辑
详细请参考 按钮执行前校验.docx

kanban

  • 看板点击事件

弹框

  • 大小设置
    • context: { ‘dialog-size’: ‘small/large/super’ }
    • context: { ‘width’: ‘80vw’, ‘height’: ‘70vh’ }
