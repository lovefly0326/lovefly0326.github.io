odoo Qweb

发表于 2021-03-04 更新于 2021-09-22 分类于 odoo12 阅读次数：

Qweb

QWeb 是 Odoo 使用的主要模板引擎，XML 表述，用于生成 HTML 片段和页面。
模板指令以t-开头
占位标签<t>，不会渲染任何元素

qweb 常用指令

模板声明指令

t-name

定义模板，置于顶层

数据输出指令

t-esc

渲染转义后的值，防止 xss

t-raw

不转义，渲染数据库中的原始值，不确定源数据安全性的情况下，不建议使用

条件指令

  • t-if
  • t-elif
  • t-else

循环指令

使用t-foreach 接收要迭代的集合（列表、对象或整数），t-as提供当前项的名称

以下列举了循环常用的变量， 注意：$as将指代传递给 t-as 的名称

变量名 描述
$as_all 正在迭代的集合
$as_value 对于列表和整数，该值与$as相同
对于对象(字典)，$as代表键，$as_value代表值
$as_index 索引值（从 0 开始）
$as_size 集合的大小（列表有效）
$as_first 布尔值
当前项是否是迭代的第一项（相当于 $as_index == 0）
$as_last 布尔值
当前项是否是迭代的最后一项（相当于 $as_index + 1 == $as_size）
$as_parity 当前迭代项索引的奇偶性，even 偶，odd 奇
$as_even 布尔值，判断是否为偶数索引
$as_odd 布尔值，判断是否为奇数索引

属性指令

  • 注意：以下$name将指代属性名称

t-att-$name

动态创建一个属性

t-attf-$name

与t-att-$name相同，但参数是一个格式字符串，常用于混合文字和非文字字符串

t-att=mapping

值是字典

t-att=pair

值是 2 个元素的数组

设置变量指令

t-set

用于设置变量名称

t-value

变量的值

  • 有t-value，值即为变量的值

  • 没有t-value，节点的内容即为变量的值

调用子模板的指令

t-call

  • 调用子模板，并传入变量

  • call 指令的主体可以是任意复杂的（不仅仅是 set 指令），其将在被调用模板中作为一个0的变量呈现

模板继承指令

t-extend

值为要继承修改的现有模板的名称(t-name 的名字)
用于更改现有模板
如果t-name与t-extend 同时存在，则新建指定名称的模板，而不会修改现有模板

t-jquery

值为 css 选择器，用于匹配现有模板中的节点内容

t-operation

对t-jquery匹配到的元素进行指定的操作

属性名 描述
append 用于添加子节点(后面)，等同于 jquery 的append()
prepend 用于添加子节点(前面)，等同于 jquery 的prepend()
before 在当前节点前添加节点(同级)，等同于 jquery 的 before()
after 在当前节点后添加节点(同级)，等同于 jquery 的 after()
inner 替换子节点，等同于 jquery 的 html()
replace 替换当前节点包括子节点，等同于 jquery 的 replaceWith()
attributes 用于添加属性，有则替换，无则添加，等同于 jquery 的 attr()
没有t-operation 通过 javascript 执行相应的逻辑，this指向t-jquery 匹配的 jquery 对象

调试指令

主要用于开发阶段的调试工作

t-log

用于打印，相当于 console.log

t-debug

模板中的断点，相当于 js 中的 debugger，在开发者模式下会进入断点

t-js

模板呈现期间执行的 jsvascript 代码，获取一个 context 上下文参数
